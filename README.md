Meu Portfólio
Este projeto é um portfólio pessoal desenvolvido utilizando React e Vite no frontend, com Tailwind CSS para o estilo, e Node.js no backend. O objetivo deste portfólio é apresentar minhas habilidades, projetos e experiências.

Tecnologias Utilizadas
Frontend:
React
Vite
Tailwind CSS
Backend:
Node.js

mysqlworkbench
